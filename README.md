## About the Project
LunaLog is an Android mobile application designed to help its users track their menstrual cycles, symptoms, mood, and personal notes. The app features a clean and accessible interface while allowing users to log in and store their information for ongoing tracking and review of their data. 

## Features
- User account creation and login with input validation, supporting email/password authentication and Google sign-in
- Initial collection of relevant user information (e.g. date of birth and average cycle length)
- Profile and settings management, allowing users to update their account details and log out
- Dashboard providing access to diary entries, mood tracking, and a period calendar
- Period calendar for logging cycle start and end dates, including retrospective entries
- Mood and symptom tracking associated with menstrual cycles, with calendar-based navigation
- Daily diary feature for journaling and logging personal notes
- Visual dashboard displaying tracked data using charts and summaries (e.g. mood trends, flow intensity, energy levels)
- Basic monthly cycle estimates based on previously logged data
- Integration of an external API to display current weather information
- Clean, minimalistic user interface following Material Design principles

## Tech Stack
- Languages: Kotlin, Java
- IDE: Android Studio
- UI & Design: Material Design principles
- Planning & Design: Wireframing
